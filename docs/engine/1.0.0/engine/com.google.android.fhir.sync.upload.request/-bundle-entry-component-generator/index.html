BundleEntryComponentGenerator

abstract class BundleEntryComponentGenerator(httpVerb: Bundle.HTTPVerb, useETagForUpload: Boolean)

Abstract class for generating Bundle.BundleEntryComponent for a Patch to be added to the Bundle based on Bundle.HTTPVerb supported by the Fhir server. Concrete implementations of the class should provide implementation of getEntryResource to provide Resource for the LocalChangeEntity. See [https://www.hl7.org/fhir/http.html#transaction] for more info regarding the supported Bundle.HTTPVerb.
